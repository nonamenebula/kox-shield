# CHANGELOG — KOX Shield

## 2026.04.14

- Добавлен экран настроек `/settings` в боте: автообновление KOX, автообновление списков, управление уведомлениями
- Бот уведомляет о новых версиях KOX Shield: кнопки «Обновить», «Не сегодня», «Не этот месяц», «Отключить»
- Бот показывает конкретные новые домены в уведомлении об обновлении списков
- Добавлена категория `supercell` — Clash Royale, Clash of Clans, Brawl Stars
- Добавлены `cursor.com`, `cursor.sh` в категорию `github-dev`
- `claude.ai` уже включён в `chatgpt-openai`
- `kox upgrade --force` для вызова из бота без интерактивного подтверждения
- По умолчанию автообновление выключено (нужно включить в `/settings`)
- Добавлены списки доменов по категориям (29 категорий, 250+ доменов и IP)
- Новые команды: `kox list-cats`, `kox list-load`, `kox list-remove`, `kox list-check`, `kox list-update`
- Telegram бот уведомляет об обновлении списков с предложением обновить
- Настраиваемые уведомления: отложить на день, на месяц или отключить
- Просмотр категорий из бота: `/listcats`

## 2026.04.13

- Версия KOX Shield теперь отображается в баннере консоли и выводе `kox status`
- Telegram бот показывает версию в `/start`, статусе и справке
